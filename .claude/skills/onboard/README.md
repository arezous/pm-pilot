# /onboard

Guides you through populating your context files so every skill has real information to work with.

## Usage

```
/onboard
```

Or just start sharing what you know. If 3+ context files are empty, the onboard workflow triggers automatically.

## What it does

1. Reads your 4 context files (company, competitors, personas, product)
2. Reports what's filled in and what's missing
3. Asks you to share info, starting with company and working outward
4. Routes your input to the right context file
5. Repeats until your context is solid

## Tips

- **Resumable.** Run `/onboard` anytime to check what's thin and fill gaps.
- **Share URLs.** Give it a company website or competitor URL and it'll research and populate context for you.
- **Dump everything.** Paste a pitch deck, brain dump, or doc. It'll sort the pieces into the right files.
- **Pair with other skills.** Once context is solid, try `/analyze-competitors` for deep dives or `/prd` to start speccing features.
